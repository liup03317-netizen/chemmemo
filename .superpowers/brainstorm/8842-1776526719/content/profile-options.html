为“档案”页增加哪些内容？

基于我们已有的数据（连胜、错误标签、方程式掌握度），这里有几个不同侧重点的设计方向：

A

方案 A：掌握度矩阵 (Progress Focus)

侧重于展示“你学会了多少”。通过进度条展示各个章节（如“氧气”、“金属”）的方程式掌握比例（未学习 / 学习中 / 已掌握）。

第八单元：金属和金属材料
40% 已掌握 | 20% 学习中
B

方案 B：薄弱点诊断 (Weakness Focus)

侧重于展示“你哪里容易错”。汇总历史错题标签（如配平错误、条件遗漏、符号错误），用雷达图或横向柱状图指出学生的具体知识盲区。

反应条件易忘 12次错误
配平错误 5次错误
C

方案 C：学习日历与动态徽章 (Habit Focus)

侧重于“坚持”。引入类似 GitHub 的学习贡献图（绿点日历），记录每天的复习量；并将现有的成就系统升级为随数据动态解锁的徽章库。

D

方案 D：全能综合看板 (Comprehensive)

将上述三者（章节进度、易错点分析、学习日历）浓缩整合到一个长列表中，向下滚动即可全面了解自己的学习画像。
